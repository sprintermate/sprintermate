# 4. Project & Sprint Management

## 4.1 Home Page – Project List

- [ ] The home page lists all registered projects as cards.
- [ ] Each card shows: project name, organization, last sprint info, and status (active/inactive).
- [ ] An "Add Project" button opens the onboarding flow.
- [ ] Clicking a project navigates to the **Sprint Selection** screen.

---

## 4.2 Sprint Selection Screen

- [ ] The **current sprint**, **previous sprint**, and **next sprint** of the selected project are listed.
- [ ] Each sprint card shows: sprint name, date range, total item count, and completion percentage.
- [ ] Clicking a sprint displays the work items inside it.
- [ ] The sprint selection screen also includes a **"Create Room"** button.
